# Acme Explorer project

# Getting started

## Build the image
```
docker-compose build 
```

## Run the containers
```
docker-compose up -d
```

## Down the containers
```
docker-compose down
```

## Manage database from MongoDB Compass
```
mongodb://acme:acme@localhost:27017/ACME_Explorer
```
